## EvadeStrike ゲーム説明


### 概要
ステージ攻略型3Dアクションゲームです。\n
コンボ攻撃とジャスト回避を活かして爽快に攻略しよう！\n


### システム
コンボ攻撃：\n
　特定の順序で攻撃を入力するとコンボ成立！\n
　強力な攻撃が発動！\n

　発動条件\n
　　弱または強攻撃を5回\n
　　強→弱→強\n
　　弱→強→強→弱\n

ジャスト回避：\n
　回避行動中に敵の攻撃がかすめるとジャスト回避発動！\n
　敵が一定時間スローになりコンボし放題！\n


### 操作(Xboxコントローラー)

移動　　：　左スティック\n
弱攻撃　：　Yボタン\n
強攻撃　：　Bボタン\n
回避　　：　Rボタン\n
ジャンプ：　Aボタン\n
