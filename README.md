# EvadeStrike ゲーム説明


## 概要
ステージ攻略型3Dアクションゲームです。  
コンボ攻撃とジャスト回避を活かして爽快に攻略しよう！  


## システム
### 　コンボ攻撃：  
　特定の順序で攻撃を入力するとコンボ成立！  
　強力な攻撃が発動！  

### 　発動条件  
　　弱または強攻撃を5回  
　　強→弱→強  
　　弱→強→強→弱  

### ジャスト回避：  
　回避行動中に敵の攻撃がかすめるとジャスト回避発動！  
　敵が一定時間スローになりコンボし放題！  


## 操作(Xboxコントローラー)

移動　　：　左スティック  
弱攻撃　：　Yボタン  
強攻撃　：　Bボタン  
回避　　：　Rボタン  
ジャンプ：　Aボタン
